In this paper, we propose a quality-aware data fusion and enhancement framework for Medical Concept Normalization (MCN). 
To the best of our knowledge, this work is the first to demonstrate that multi-source dataset fusion can substantially improve MCN performance, and we validated this through extensive experiments on social media health datasets.
